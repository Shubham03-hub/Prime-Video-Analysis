# Prime-Video-Analysis
An interactive data analytics dashboard built to analyze content distribution, ratings, genres, and trends on Prime Video.

This project demonstrates data cleaning, exploratory data analysis (EDA), and dashboard development skills using real-world streaming data.

Project Objective-
The goal of this project is to Analyze Prime Video content trends, Understand distribution by genre, country, and year, Examine ratings and content types (Movies vs TV Shows)
Build an interactive dashboard for insights.
This project simulates a real-world streaming analytics use case.
